Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/wannier90_input>`_ with uv:

.. code-block:: console

    $ uv pip install wannier90_input

or with pip:

.. code-block:: console

    $ python3 -m pip install wannier90_input

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/elinscott/wannier90-input.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/elinscott/wannier90-input.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/elinscott/wannier90-input.git
    $ cd wannier90-input
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
